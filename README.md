# buzzWORD

Repo for Junction 2017 project for team buzzWORD

## Init Front

`cd front`

`npm i`

`npm start`

Front runs in port 8080

## Init Api

`cd api`

`npm i`

`pip install numpy` `pip install sklearn` `pip install pymongo`

`cd pythonScripts` `python import_flight_data.py`

`npm run build`

`npm start`

Api runs in port 8000
